# 回测任务控制与汇总施工计划

本批实现 V3.3 第 25.14.6 至 25.14.9 节中的回测任务列表、汇总、暂停、继续、取消、失败重试和重新运行。范围限定为当前已支持的单股回测，不实现导出、监控列表回测或全市场回测。

风险等级为 L3。回测模块继续独占任务、子项、结果和控制命令；通用 Job 只代表一次执行尝试。任务使用单调递增的执行代数拒绝迟到旧任务，控制命令保存幂等事实，状态、审计、事件和新执行任务在同一事务提交。

暂停和取消采用安全点协作：外部策略结果和公司行动结果先冻结，再响应控制；最终结果保存前再次检查任务状态。失败重试复用可证明完整的冻结快照，预测结果未知时拒绝重试。重新运行创建新任务并保留旧任务和旧结果。

公开接口新增分页列表、汇总、子项列表及五个控制接口。所有写操作要求登录、可信写身份、确认、原因和幂等键。汇总只展示单股进度、错误和单股指标，不计算组合收益。

完成条件：合法和非法状态转换、重复命令、旧执行代数、暂停/取消竞态、失败恢复、重新运行隔离、权限输入和前端重复提交均有验证；模块接入后执行一次后端、前端、迁移和服务器容器验收。
